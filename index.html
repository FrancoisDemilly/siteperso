Projets

Recipe Box

  • CRUD app basée on Create-React-App. Déployée on Now.sh
  • Créée with HTML5, CSS3, ReactJS
  • Les recettes sont conservées sur Firebase. La synchronisation de l’app avec la base de données est faite par Re-base et la fonction syncState

Pomodoro Clock

  • Créée avec HTML5, CSS3 et Javascript
  • Les ensembles de données du DOM sont stockés dans un tableau, sur lequel une fonction forEach est appliquée. Pour chaque élément cliqué, des fonctions d'assistance sont déclenchées pour gérer la saisie utilisateur et démarrer le compte à rebours de l'horloge et le remplissage

Calculator

  • Créée avec HTML5, CSS3 et Javascript
  • Les valeurs et les ensembles de données du DOM sont stockés dans un tableau, sur lequel une fonction forEach est appliquée. Pour chaque élément cliqué, des fonctions d'assistance sont déclenchées pour gérer l'entrée utilisateur
  • La fonction eval () est utilisée pour les calculs

Simon Game

  • Créée avec HTML5, CSS3 et Javascript
  • Simon Game est un jeu dont le but est que l'utilisateur répète le modèle montré avec le son par le programme. À chaque tour, une nouvelle étape est ajoutée au motif

TicTacToe Game

  • Créée avec HTML5, CSS3 et Javascript
  • Tic Tac Toe est un jeu dont le but est pour l'utilisateur de battre l'adversaire en tirant 3 marques consécutives
  • Le choix de l'ordinateur est aléatoire, l'algorithme Minimax n'a pas été utilisé pour le jeu
